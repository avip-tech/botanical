# 🌿 Botanical - Premium Perfume Marketplace

A luxurious, modern perfume marketplace website with a personalized recommendation system.

## 🎨 Design Features

- **Premium Color Scheme**: Black, Gold, and Cream
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Modern UI**: Smooth animations and elegant transitions
- **Premium Feel**: Luxury aesthetic throughout

## ✨ Features

### 1. **Product Showcase**
   - Featured collection display
   - Product details with pricing
   - Add to cart functionality

### 2. **Personalized Recommendation Quiz**
   - Skin type detection (Oily, Dry, Mixed, Sensitive)
   - Occasion matching (Formal, Casual, Party, Office, Wedding, Evening)
   - Fragrance preference (Floral, Woody, Citrus, Oriental, Fresh)
   - AI-powered recommendations

### 3. **User Features**
   - User authentication (Login/Signup)
   - Shopping cart system
   - Product reviews (coming soon)
   - Wishlist functionality (coming soon)

## 🚀 Getting Started

### Live Website
Your website is live at: **https://avip-tech.github.io/botanical**

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Local Installation

1. Clone the repository:
```bash
git clone https://github.com/avip-tech/botanical.git
cd botanical
```

2. Open `index.html` in your browser:
```bash
open index.html
```

## 🔄 How the Quiz Works

1. User selects skin type
2. User selects occasion
3. User selects fragrance preference
4. System recommends best-matching perfume
5. User can add recommended product to cart
